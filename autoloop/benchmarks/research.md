# Scientific Research Benchmarks / 科研基准测试

## Task 1: 文献综述结构化
Given 10 paper abstracts on a topic, generate:
- Thematic grouping (not chronological)
- Consensus vs controversy mapping
- Methodology comparison matrix
- Research gaps identification
- Suggested next research questions

评估标准: 主题分类准确度、矛盾识别率、缺口发现的有价值程度

## Task 2: 实验设计审查
Given a proposed experimental protocol, review for:
- Statistical power and sample size adequacy
- Control group design
- Confounding variables
- Reproducibility concerns
- Ethical considerations

评估标准: 问题识别完整度、建议的可操作性、是否遗漏关键风险

## Task 3: 数据分析与可视化代码
Given a dataset description and research question:
- Write Python/R analysis code
- Include appropriate statistical tests
- Generate publication-ready visualizations
- Interpret results in context of the research question
- Flag limitations and caveats

评估标准: 代码正确性、统计方法选择合理性、可视化质量、结果解读深度
