# Pharma & Biotech Benchmarks / 医药行业基准测试

## Task 1: 临床试验数据解读
Given a Phase II/III clinical trial readout:
- Extract primary and secondary endpoints
- Evaluate statistical significance (p-values, confidence intervals)
- Compare with competitor trials in same indication
- Assess safety signal severity
- Summarize implications for regulatory filing

评估标准: 数据提取准确性、统计解读正确性、竞品对比完整度

## Task 2: 药物管线分析
Given a biotech company's pipeline:
- Map each program by phase, indication, mechanism of action
- Estimate probability of success by phase
- Identify competitive landscape per indication
- Calculate risk-adjusted NPV for top 3 assets
- Flag upcoming catalysts (data readouts, FDA dates)

评估标准: 管线覆盖完整度、成功概率估算合理性、催化剂识别准确度

## Task 3: 监管文件审查
Given an FDA complete response letter (CRL) or advisory committee briefing:
- Identify key regulatory concerns
- Map concerns to addressable vs fundamental issues
- Compare with precedent decisions in same therapeutic area
- Assess resubmission timeline and probability
- Draft response strategy outline

评估标准: 关键问题识别率、先例引用准确性、策略建议可行性
